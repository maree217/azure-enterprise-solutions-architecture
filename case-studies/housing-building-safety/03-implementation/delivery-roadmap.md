# Delivery Roadmap

## Meridian Housing Group - Programme Timeline

*Roadmap aligned to the [Three-Layer Framework](../../../three-layer-overlay/README.md) phased delivery approach.*

---

## Programme Overview

The Building Safety Platform programme spans 18 months from assessment through full rollout, with value delivered incrementally at each phase.

**Programme Duration:** 18 months
**Start:** January 2024
**Full Go-Live:** June 2025

## Dependencies and Assumptions

### External Dependencies

| ID | Dependency | Owner | Impact | Mitigation |
|----|------------|-------|--------|------------|
| D1 | Northgate DB read access | IT/Vendor | Critical path | Early engagement, 4-week lead |
| D2 | VPN hardware procurement | IT | Phase 1 timeline | Order in W13 |
| D3 | Power Platform licences | Finance | Phase 2 timeline | Budget approval by W20 |
| D4 | Gas-Track API access | Contractor | Phase 2 data | API key request W24 |
| D5 | Pen test scheduling | External | Go-live | Book 6 weeks ahead |

### Assumptions

| ID | Assumption | Risk if False |
|----|------------|---------------|
| A1 | Northgate vendor cooperative | Delays to integration |
| A2 | Data quality as assessed | Extended cleansing phase |
| A3 | Partner resources available | Delayed start |
| A4 | No major M365 changes | Rework required |
| A5 | Users available for UAT | Delayed go-live |
