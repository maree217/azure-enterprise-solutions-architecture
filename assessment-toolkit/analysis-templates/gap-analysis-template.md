## Gap Classification Framework

### Severity Levels

| Level | Description | Response Time |
|-------|-------------|---------------|
| **Critical** | Immediate security risk, compliance violation, or service impact | < 1 week |
| **High** | Significant risk or deviation from best practices | 1-4 weeks |
| **Medium** | Improvement opportunity with moderate impact | 1-3 months |
| **Low** | Nice-to-have improvement, optimization opportunity | 3-6 months |

### Gap Categories

| Category | Description |
|----------|-------------|
| **Security** | Identity, network, data protection, compliance |
| **Reliability** | Availability, disaster recovery, backup |
| **Performance** | Scaling, optimization, latency |
| **Cost** | Waste, rightsizing, optimization |
| **Operations** | Monitoring, automation, documentation |
| **Governance** | Policy, compliance, standards |
